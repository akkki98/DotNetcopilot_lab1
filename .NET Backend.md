# Lab: Building a .NET Backend API with GitHub Copilot

## **Lab Scenario**
You are working as a backend developer for **QuickKart**, an online retail platform.  
Your task is to build a **Product Management API** using **ASP.NET Core Web API**.  
You will use **GitHub Copilot** to speed up development in four key backend areas:

1. Generate REST API endpoints  
2. Implement database operations using EF Core  
3. Add structured error handling and logging  
4. Write unit tests using xUnit and Moq  

---

## **Prerequisites**
- .NET 8 SDK  
- Visual Studio / VS Code  
- SQL Server LocalDB or SQLite  
- GitHub Copilot enabled  

---

## **Lab Objectives**
By the end of this lab, you will:

- Build API endpoints using Copilot prompts  
- Scaffold EF Core models, DbContext, and repository with Copilot  
- Add centralized error handling and logs  
- Generate unit tests using Copilot
